# ai-naming

# setup

```
$ cp .env.template .env
$ vi .env
$ make build-linux
```

# how to use

```
$ naming
```

```
? Choose a type:  [Use arrows to move, type to filter]
> 関数
  変数
```

```
? Enter an overview: 処理の内容を具体的に記述する
```
